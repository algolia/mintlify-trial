---
title: InstantSearch
description: Open source libraries for building search and discovery interfaces.
---

InstantSearch lets you create fast and seamless search experiences using Algolia.
It provides a range of customization APIs so you can tailor the search interface to your specific needs.
These APIs let you modify the appearance, behavior, and functionality of the search components, ensuring a personalized and engaging user experience.
