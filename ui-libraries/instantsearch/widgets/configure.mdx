## About this widget 

The `configure` widget lets you provide raw search parameters to the Algolia API without rendering anything.

Any props you add to this widget is forwarded to Algolia. For more information on the different parameters you can set, have a look at the [search parameters API reference](/doc/api-reference/search-api-parameters/).

## Customize the UI

<Tabs>
  <Tab title="JS">
    If you want to create your own UI of the `configure` widget, you can use connectors.

    To use `connectConfigure`, you can import it with the declaration relevant to how you installed InstantSearch.js.
    
    <Steps>
      <Step title="Create a render function">
        ```js
const renderConfigure = (renderOptions, isFirstRender) => {
  // Rendering logic
};
```

      </Step>
      <Step title="Create the custom widget">
        ```js
import { connectConfigure } from 'instantsearch.js/es/connectors';

const customConfigure = connectConfigure(
  renderConfigure
);
```

      </Step>
      <Step title="Instantiate">
        ```js
search.addWidgets([
  customConfigure({
    // instance params
  })
]);
```

      </Step>
    </Steps>
  </Tab>
  <Tab title="React">
    React InstantSearch let you create your own UI for the `configure` widget with `useConfigure`. Hooks provide APIs to access the widget state and interact with InstantSearch.

    The `useConfigure` Hook accepts parameters and returns APIs. It must be used inside the `<InstantSearch>` component.
    
    <Steps>
      <Step title="Create a render function">
        ```js
import { useConfigure } from 'react-instantsearch';

function CustomConfigure(props) {
  const { refine } = useConfigure(props);

  return null;
}
```

      </Step>
      <Step title="Create the custom widget">
        ```js
<CustomConfigure {...searchParameters} />
```

      </Step>
      <Step title="Instantiate">
        ```js
search.addWidgets([
  customConfigure({
    // instance params
  })
]);
```

      </Step>
    </Steps>
  </Tab>
  <Tab title="Vue">
    If you want to create your own UI of the `configure` widget, you can use slots.
  </Tab>
