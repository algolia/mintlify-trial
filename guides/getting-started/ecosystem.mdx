You can choose from a set of libraries and tools to help connect your data and configure Algolia for your site or app.
Algolia supports several programming languages and provides integrations for popular frameworks and ecommerce platforms.

<Note>
    Discover code samples and community-built integrations in the [Algolia Code Exchange](https://www.algolia.com/developers/code-exchange/).
</Note>

## Algolia dashboard

In the [dashboard](https://dashboard.algolia.com/dashboard/) you can configure most aspects of Algolia's products and features.

## Algolia CLI

The [Algolia CLI](https://www.algolia.com/doc/tools/cli/get-started/overview/) lets you interact with Algolia's APIs from your terminal, without code.

## API clients

Algolia's API clients provide convenient methods for interacting with Algolia's APIs,
and add features such as a retry strategy, or batching uploads.

## REST API

While you can carry out any indexing, search, and configuration-related task with the [REST API](/api-reference/overview/),
you should use an API client or another integration.

<Warning>
    You should only use the REST API to develop new API clients.
    Using the REST API directly isn't covered by Algolia's [SLA](https://www.algolia.com/policies/sla/) terms.
</Warning>

## Frontend libraries

Algolia offers UI libraries with ready-to-use and customizable components.

### InstantSearch

InstantSearch lets you build user interfaces for search and discovery from ready-made components (widgets).
InstantSearch also gives you the tools to build your own widgets.
InstantSearch is available in these different _flavors_:

import InstantSearch from '/snippets/instantsearch.mdx';

<InstantSearch />

### Autocomplete

The [Autocomplete UI library](https://www.algolia.com/doc/ui-libraries/autocomplete/introduction/what-is-autocomplete/) lets you build accessible, search-as-you-type user interfaces that you can integrate anywhere on your site or app.

### UI design kit

If you want to design your next search and discovery experience,
you can use Algolia's [UI Design Kit (Figma)](https://www.algolia.com/doc/guides/building-search-ui/resources/ui-kit/js/).

### DocSearch

DocSearch is the easiest way to add search to a technical documentation site.
It's [free for such sites](https://docsearch.algolia.com/docs/who-can-apply/).

Find examples of DocSearch for popular site builders, such as Docusaurus or VuePress,
in the [Code Exchange](https://www.algolia.com/developers/code-exchange/?refinementList%5Bproduct_features%5D%5B0%5D=Algolia%20Crawler).

To get started, [apply online](https://docsearch.algolia.com/apply/).
